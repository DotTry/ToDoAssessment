# SogetiAssessment
Submission for Todo ReactJS, Cordova webapp



```markdown
# Installation

1. git clone https://github.com/DotTry/ToDoAssessment/ 

2. npm install


#Running on browser: npm start


Android: npm run cordova-build-prod-android

iOS: npm run cordova-build-prod-ios


Unit Tests: npm test
   ```
